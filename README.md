# 🌌 Isara Ekanayake | Software Engineer Portfolio

**[🔴 Live Demo: View Portfolio](https://isarase-portfolio.surge.sh/)**

A modern, high-performance portfolio website built to showcase software engineering projects, skills, and experience. Designed with a premium dark-space aesthetic, featuring interactive 3D astronomy animations and a buttery-smooth scrolling experience.


## ✨ Features

- **Interactive 3D Animations**: Powered by Three.js. Features a majestic rotating galaxy in the hero section and a celestial ringed planet in the globe section, complete with parallax mouse interactions.
- **Buttery Smooth Scrolling**: Integrated with [Lenis](https://lenis.studiofreight.com/) for a fluid, physics-based scrolling experience.
- **Premium Typography**: Utilizes `Space Grotesk` for headings and `Plus Jakarta Sans` for body text to perfectly complement the modern space theme.
- **Custom UI Elements**: Custom cursor interactions, micro-animations on hover states, and a dynamic glassmorphism navigation bar.
- **Fully Responsive**: Optimized for seamless viewing across mobile, tablet, and desktop displays.
- **Zero-Build Architecture**: Built entirely with Vanilla HTML, CSS, and JavaScript. No build steps required.

## 🚀 Tech Stack

- **Frontend Structure**: HTML5
- **Styling**: Vanilla CSS3 (CSS Variables, Flexbox, CSS Grid)
- **Interactivity**: Vanilla JavaScript (ES6+)
- **3D Graphics**: [Three.js](https://threejs.org/)
- **Smooth Scroll**: [Lenis](https://github.com/studio-freight/lenis)

## 💻 Running Locally

Since this project uses vanilla web technologies without any bundlers, it's incredibly simple to run locally.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/IsaraSE/Portfolio.git
   cd Portfolio
   ```

2. **Serve the project:**
   You can use any local HTTP server to view the website. Using Python's built-in server is the quickest way:
   ```bash
   python3 -m http.server 8080
   ```
   *(Alternatively, you can use the Live Server extension in VS Code).*

3. **View in browser:**
   Navigate to `http://localhost:8080` in your web browser.
